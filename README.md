# Project Setup

## Prerequisites

**Make sure you have the following installed:**

- Node.js (v18 or later recommended)
- npm or yarn

## Clone the Repository

- **git clone https://github.com/RishabhJain999/React-Query-Pagination.git**

## Install Dependencies

- **npm install**

## This will install all required packages including:

- React

- TypeScript

- React Query

- Tailwind CSS

- Axios

## Start the Development Server

- **npm run dev**

## The application will start on:

- **http://localhost:5173**

# React Pagination using React Query

    This project demonstrates server-side pagination in a React web application using
    @tanstack/react-query v5 with real API data.

The app fetches **posts** from the JSONPlaceholder API and implements pagination with:

- cached pages
- smooth transitions
- prefetching
- proper loading and error handling

---

## Features

- Paginated list of posts using a real API
- Previous / Next navigation
- Cached pages using React Query
- Prefetching of next page for smoother UX
- Skeleton loader while data loads
- Error and empty state handling
- Clean, reusable component structure
- Type-safe code with TypeScript
- Styling using Tailwind CSS

---

## Tech Stack

- **React 19**
- **TypeScript**
- **@tanstack/react-query v5**
- **Axios**
